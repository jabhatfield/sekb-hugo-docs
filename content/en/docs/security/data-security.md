## Requirements

{{% pageinfo %}}

* Encrypt passwords.
* Store credentials safely.
* Restrict all access.

{{% /pageinfo %}}

## Examples

* Encryption:
  * [MD5](https://en.wikipedia.org/wiki/MD5)
* Credential storage:
  * [Vault](https://www.vaultproject.io/)
* Restrict access:
  * [MySQL roles](https://dev.mysql.com/doc/refman/8.0/en/roles.html)
